# Drum Kit Web App

An interactive drum kit web application that produces real-time drum sounds using keyboard keys or mouse clicks. Built with HTML, CSS, and vanilla JavaScript.

## Features

* Play drum sounds using keyboard input
* Click buttons to trigger drum beats
* Real-time audio playback
* Button animations for visual feedback
* Responsive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Use

1. Open the webpage
2. Press the keys displayed on the drum buttons
3. Or click the buttons with your mouse
4. Enjoy playing drums
